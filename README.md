# Visit Gem City

A beautiful tourism website for Ratnapura gem city tours in Sri Lanka.

## Project Structure

- `frontend/` - Vue.js application with Tailwind CSS
- `backend/` - Future API development (placeholder)

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Deployment

This project is deployed on Netlify with automatic builds from the main branch.

## Environment Variables

Create `.env` file in frontend directory:
```
VITE_FORMSPREE_ID=your_formspree_id
```

## Features

- Responsive design
- Dark mode support
- Contact form with Formspree
- Modern UI with Tailwind CSS
